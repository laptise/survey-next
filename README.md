# Survey-next

패키지 관리 yarn 사용

### 개요

설문 또는 질문작성가능 웹앱

### 기술 스택

- React.js(Next.js)
- Typescript
- MySql
- etc..

### 개발환경 구축 및 테스트

#### 의존라이브러리 설치 (yarn이 설치 되어있어야함)

`yarn`

#### 디버그 모드에서 실행

`yarn dev`

#### 빌드

`yarn build`

#### 서브

`yarn start`

## 과제

- 인증 정보 없을 경우 리다이렉트를 어떻게 할지
- 인증을 어떻게 할건지
- api 헤더에 무엇을 위해 뭘 담을지
